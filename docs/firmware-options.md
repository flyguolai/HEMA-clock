# 固件选项

以下为社区已开发的成熟固件，可直接烧录使用。

## 固件对比

### 2.13 寸屏幕

| 固件名称 | 作者 | 功能 | 特点 |
|---------|------|------|------|
| 盒马213时钟25版 | 飘 | 14种时钟 + 7种日历界面 | 支持正点/斜点IO自识别 |
| DA14585竖屏时钟日历 | 随缘轻舞 | 时钟 + 日历 + 温度 + 倒计时 | 竖屏显示，适合挂件 |
| TSL0922 固件 | tsl0922 | 日历 + 时钟 | 蓝牙同步，支持多种驱动板 |
| HAT18 固件 | 随缘轻舞 | 时钟日历 | 最新版，功能丰富 |

### 4.2 寸屏幕

| 固件名称 | 作者 | 功能 | 特点 |

## 固件获取

- **墨韵时光网站**: https://www.myetoys.com/firmware
- **QQ群**: 893314999（墨水屏时钟DIY）
- 部分固件需要激活（付费 5 元），支持作者持续开发

## 固件选择建议

1. **2.13 寸 + 想要最多功能** → 盒马213时钟25版
2. **2.13 寸 + 竖屏挂件** → DA14585竖屏时钟日历
3. **4.2 寸** → DA14585 4.2寸时钟日历
4. **想学习 BLE 同步** → TSL0922 固件
5. **多屏联动** → 四联屏同步时钟
