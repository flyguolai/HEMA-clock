# 硬件资料

## 主控芯片：DA14585

- **厂商**: Renesas (原 Dialog Semiconductor)
- **类型**: BLE 5.0 SoC
- **Flash**: 64KB OTP + 外部 SPI Flash
- **RAM**: 84KB
- **封装**: QFN-40 (5x5mm) / WLCSP
- **特点**: 超低功耗，适用于电池供电设备

### 启动模式

DA14585 支持两种启动方式：
1. **一次启动 (XIP)**: 固件直接写入 OTP，无法更改
2. **二次启动 (Boot from SPI Flash)**: OTP 中烧录 Bootloader，APP 从外部 SPI Flash 加载（可多次刷写）

盒马价签一般采用二次启动方式，OTP 中已烧录 Bootloader，固件存储在外部 SPI Flash 中。

## 墨水屏

### 屏幕型号

盒马 2.13 寸价签常见屏幕型号：
- **E029A01** — 2.13" 黑白，250×122 分辨率
- 驱动 IC: SSD1680（或兼容型号）

### 屏幕参数

| 参数 | 值 |
|------|-----|
| 尺寸 | 2.13 英寸 |
| 分辨率 | 250 × 122 像素 |
| 颜色 | 黑白 (部分为黑白红三色) |
| 驱动 IC | SSD1680 |
| 接口 | SPI |
| 刷新方式 | 全刷 / 局刷 |

### 4.2 寸版本

部分盒马价签使用 4.2 寸屏，同样基于 DA14585 主控。

## PCB 板载资源

- DA14585 主控
- 外部 SPI Flash（存储固件）
- 墨水屏 FPC 连接器
- 电池接口
- 预留 LED 焊盘 (D1, 连接 P2_5)
- 预留调试焊盘（SWD、UART）
