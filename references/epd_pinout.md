# EPD Board Pin Mapping

## DA14585 ↔ 连接器 完整引脚映射

| 连接器脚 | DA14585 | 功能 | 说明 |
|:---:|:---:|------|------|
| 1 | P2_2 | **MOSI** | SPI 数据输出到 EPD |
| 2 | ── | **VCOM** | EPD 参考电压，~100mV→0 周期 |
| 3 | GND | **GND** | 地 |
| 4 | ── | NC | 断路 |
| 5 | ── | **VG** | 升压电容，5V→0→20mV 周期 |
| 6 | ── | NC | 断路 |
| 7 | ── | NC | 断路 |
| 8 | GND | **GND** | 地 |
| 9 | P1_1 | **BUSY** | EPD 忙检测，平时 3.3V，EPD 处理时拉到 0V |
| 10 | P1_0 | **ADC** | 电池电压采样，基本保持 3.3V |
| 11 | P0_7 | **RST** | EPD 复位，平时 3.3V，复位时 0V→1.9V 脉冲 |
| ── | ── | **LED** | 板上未见 LED，固件特征值可能未实际使用 |
| 12 | P2_1 | **CS** | SPI 片选，平时 3.3V，刷新时拉到 1.8V |
| 13 | P0_1 | **DC** | EPD 数据/命令，周期 3.3V→0短→3.3V→0长 |
| 14 | P2_0 | **SCK** | SPI 时钟 |
| 15 | ── | **VCC** | 3.3V 供电 |
| 16 | ── | **VCC** | 3.3V 供电 |
| 17 | GND | **GND** | 地 |
| 18 | GND/NC | ── | 地/断路 |
| 19 | GND/NC | ── | 地/断路 |
| 20 | GND/NC | ── | 地/断路 |
| 21 | ── | **VGH** | EPD 正高压 +18V (boost 输出) |
| 22 | GND/NC | ── | 地/断路 |
| 23 | ── | **VGL** | EPD 负高压 -18V (boost 输出) |
| 24 | GND/NC | ── | 地/断路 |

## 非连接器引脚

| DA14585 | 功能 | 说明 |
|:---:|------|------|
| P2_3 | 电源使能 | boost 升压电路开关控制 |
| P0_5 | UART TX | 调试串口发送 |
| P0_4 | UART RX | 调试串口接收 |

## EPD SPI 总线

```
P2_0 → SCK
P2_1 → CS
P2_2 → MOSI
P2_3 → 电源使能 (非 SPI，单独控制 boost)
```

## EPD 电源

```
VCC  (3.3V)  → pin 15, 16
VGH  (+18V)  → pin 21
VGL  (-18V)  → pin 23
VCOM (~0V)   → pin 2
VG   (5V)    → pin 5
```
