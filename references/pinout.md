# 盒马 2.13 寸价签引脚定义

## 调试接口

| 焊盘 | 功能 | 说明 |
|------|------|------|
| TP4 | RST | 复位引脚（高电平复位） |
| TP2 / TP10 | GND | 地线 |
| TP6 | SWDIO | SWD 数据线 |
| TP5 | SWCLK | SWD 时钟线 |
| TP1 / TP9 | VBAT (3V) | 电池电压 |
| TP3 | VDD | 电源 |

## UART 串口

| 焊盘 | 引脚 | 功能 |
|------|------|------|
| TP8 | P0_5 | UART RX |
| TP7 | P0_4 | UART TX |

## 墨水屏 SPI 接口

| 信号 | DA14585 引脚 | 说明 |
|------|-------------|------|
| BUSY | P2_0 | 屏幕忙信号 |
| RST | P0_7 | 屏幕复位 |
| DC | P0_5 | 数据/命令选择 |
| CS | P2_1 | 片选 |
| SDI (MOSI) | P0_6 | SPI 数据输入 |
| SCK (SCLK) | P0_0 | SPI 时钟 |
| EPD_POWER | P2_3 | 屏幕电源控制 |

## LED

| 元件 | DA14585 引脚 | 说明 |
|------|-------------|------|
| D1 | P2_5 | 板载 LED（需焊接 R4 + LED） |

## J-Link 连接参考

```
J-Link 引脚    价签焊盘
──────────────────────
VCC      ←→    TP1/TP9 (VBAT 3V)
GND      ←→    TP2/TP10 (GND)
SWDIO    ←→    TP6
SWCLK    ←→    TP5
RESET    ←→    TP4 (可选)
```

## 注意事项

- 部分价签的 OTP 可能已被写入，需要确认启动模式
- 焊接时注意不要短路相邻焊盘
- RST 引脚为高电平复位，连接时可不接
